# badminton-scheduler

🏸 羽球即時動態排程系統 (Badminton Real-time Scheduler)
這是一個專為羽球愛好者開發的智慧排程系統，旨在解決多人聚會時「排程不公」、「實力不均」以及「場地空等」的實務問題。

🌟 核心功能
實力均衡對戰 (Skill Balancing)：採用窮舉法與權重計算，自動在四人中匹配出能力值總和最接近的對戰組合，並優先配置「混雙」對抗。

即時動態補位 (Live Substitution)：支援單場結算功能。當某一場地結束時，系統會立即從休息區挑選等待權重最高（等最久）的四位球員進行補位，無需全場空等。

動態能力分級 (Dynamic Leveling)：基於勝負結果自動調整球員能力值（贏家 +0.1, 輸家 -0.1），確保隨著場次增加，排程精準度也隨之提升。

多端即時同步：基於 Streamlit 框架開發，球友可透過手機網頁即時查閱當前對戰與休息名單。

🛠️ 技術架構
開發語言：Python

前端框架：Streamlit (支援響應式網頁設計)

資料儲存：JSON (本地持久化)

演算法：多目標最佳化 (組合優化、權重佇列管理)

🚀 快速開始
安裝需求：pip install streamlit

啟動系統：streamlit run app.py

部署：支援一鍵部署至 Streamlit Cloud。

🔒 權限說明
系統設有管理員密碼保護機制。一般使用者僅具備「查閱」權限；管理員登入後可進行「產生排程」、「紀錄勝負」及「球員管理」等操作。

身為一名研究機器視覺與系統控制的工程師，我將系統控制與模擬的精神融入了這個排程器中。雖然這是我第一次嘗試將 Python 腳本網頁化並部署至雲端，但看著演算法成功在現實場景中運作，並為球友們帶來便利，是非常有成就感的開發體驗。
